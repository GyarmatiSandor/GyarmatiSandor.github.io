# GyarmatiSandor.github.io

Personal website and portfolio showcasing CV and projects.

## Overview
This is a basic GitHub Pages site featuring:
- Personal introduction
- Complete CV section (education, work experience, skills)
- Project showcase
- Contact information

## Customization
To personalize this site with your information:

1. Open `index.html` and replace the placeholder text:
   - Update personal information in the CV section
   - Add your actual education and work experience
   - Update skills to match your expertise
   - Add your real projects with descriptions and links
   - Update contact links with your actual email and social profiles

2. (Optional) Modify `styles.css` to change colors, fonts, or layout

## Local Testing
To preview the site locally:
```bash
python3 -m http.server 8080
```
Then visit `http://localhost:8080` in your browser.

## Deployment
This site is automatically deployed via GitHub Pages when changes are pushed to the main branch.
